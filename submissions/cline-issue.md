**Why should this server be added?**

Famulor is an AI voice agent platform (GDPR-compliant, EU-hosted) used by businesses to run AI phone assistants. The MCP server exposes 75 tools covering the full platform API: create and configure voice assistants, make outbound calls, run campaigns, manage leads, knowledge bases (RAG), WhatsApp, SMS, phone numbers and SIP trunks.

With Cline, developers can build and operate complete voice-agent setups conversationally — e.g. "Create a German receptionist assistant for a dental practice, add the website as a knowledge base, and start a test call."

Two install options:
- Remote (recommended): `https://mcp.famulor.io/mcp` — streamable HTTP with OAuth 2.1/PKCE, no local setup
- Local stdio: `npx -y famulor-mcp` with `FAMULOR_API_KEY` env var

Also listed in the official MCP registry as `io.famulor/famulor-mcp`.
