# Famulor MCP Server

AI voice agent platform — create phone assistants, make calls, run outbound campaigns, manage leads, knowledge bases (RAG), WhatsApp, SMS and SIP trunks. 75 tools covering the full Famulor API.

- Remote endpoint: `https://mcp.famulor.io/mcp` (streamable HTTP, OAuth 2.1 + PKCE)
- Docs: https://docs.famulor.io
- Setup guide: https://www.famulor.io/feature/mcp-connector
- Source: https://github.com/bekservice/Famulor-MCP
- Get an API key: https://app.famulor.de → API Keys
